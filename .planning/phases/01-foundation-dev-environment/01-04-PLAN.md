## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-SC | Tampering | npm installs | mitigate | RESEARCH legitimacy audit + blocking human checkpoint (Task 1); shadcn official registry only (UI-SPEC Registry Safety) |
| T-01-13 | Elevation of Privilege (V3) | proxy.ts treated as auth authority | mitigate | proxy.ts does cookie-PRESENCE redirects only; the API verifies signatures on every request (RESEARCH Architectural Responsibility Map) |
| T-01-14 | Information Disclosure | tokens readable by page JS | mitigate | httpOnly cookies; client.ts never touches token values; React default escaping, no dangerouslySetInnerHTML |
| T-01-15 | Tampering | cross-origin cookie/CSRF exposure | mitigate | rewrites make all API calls first-party; SameSite=Lax holds (RESEARCH Pattern 4 / A3) |
</threat_model>
