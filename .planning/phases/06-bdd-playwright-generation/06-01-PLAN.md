## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| LLM gateway → generation | LLM-emitted gherkin + then_refs (incl. edge_type/element_key) cross into the gate + Cypher resolution — untrusted |
| flow/page text → generation prompt | KG-derived flow/page text fed into the generation prompt — treat as untrusted input |
| generation → Postgres | a draft row is written only after both gates pass |
| gate → Neo4j | read-only existence Cypher; no writes |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
