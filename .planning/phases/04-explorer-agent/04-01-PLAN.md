| explorer → Neo4j | page-derived strings written as node properties |
| explorer → Postgres checkpoints | psycopg3 pool writes run state |
| target creds → explorer | decrypted credentials in memory during a run |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04-01 | Tampering | Cypher writes in nodes.persist_to_neo4j | mitigate | Parameterized Cypher only — never f-string page-derived text (Phase-3 T-03-05 invariant) |
| T-04-02 | Info Disclosure | decrypted target creds | mitigate | Creds ONLY via target_service.get_decrypted_credentials (single decrypt surface); never logged, never written to a node |
